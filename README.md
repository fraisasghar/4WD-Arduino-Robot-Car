##Introduction
Hi! This is my 4WD Robot Car project. I built it to see how computer code can move real-world objects. It is a four-wheel-drive car that I can control with my phone.

##Abstract
I used a clear plastic frame and four small motors to build this car. A part called the L298N driver acts like a middleman to control the wheels. I used the Arduino software to write the code. Now, I can send signals from my mobile phone via Bluetooth to drive it around.
